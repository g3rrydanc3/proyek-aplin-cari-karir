## Installation
1. Import database from `job_comer.sql`
2. Edit config.php
```php
$db_server
> Your MySQL Server
$db_username
> Your MySQL Username
$db_password
> Your MySQL Password
$db_dbname
> Your MySQL Database Name
$config_folder
> Your Website (in folder), if not just leave it empty
```
